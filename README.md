# Getting Started

To run this project you need a .env file with those variables:

DATABASE_URL
PORT
AWS_BUCKET_NAME
AWS_BUCKET_REGION
AWS_ACCESS_KEY
AWS_SECRET_ACCESS_KEY

- Clone this project (Back)
- npm install
- npm run server
- Clone [front](https://github.com/yoannesbourg/Product-catalog-front)
- npm install
- npm run start
